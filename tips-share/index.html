# 在线 VSCode 查看 GitHub 项目

比如你要看 Vue3 的源码，它在 Github 上的源码地址是 https://github.com/vuejs/coreopen in new window，如果你想使用网页版 VSCode 查看它，你只需在 https://github 后插入 1s，修改为 https://github1s.com/vuejs/coreopen in new window 即可。
